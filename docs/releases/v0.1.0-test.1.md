# Game Sprite Forge v0.1.0-test.1

This is the first public test release of Game Sprite Forge.

Game Sprite Forge is a local macOS workbench for converting videos, PNG sequences, sprite sheets, and `.gsfpack` folders into game-ready 2D sprite animation assets.

## Download

macOS Apple Silicon:

```text
Game.Sprite.Forge_0.1.0_aarch64.dmg
```

SHA-256:

```text
0a5d18c3d1c8df79ba73e617ed972241cc2208b553227fda56ec124e90b5cc42
```

## Highlights

- Local import workflow for video, PNG sequences, sprite sheets, and `.gsfpack`.
- Fixed-grid and transparent-gutter sprite sheet splitting.
- Background processing, frame normalization, quality checks, and export readiness.
- PNG frames, sprite sheet, manifest, atlas metadata, preview material, and `.gsfpack` export.
- Local pack validation and re-import flow.
- Godot helper sample.

## Notes

- This is a test release; expect rough edges.
- Apple Silicon macOS is the primary packaged target for this release.
- Video workflows require `ffmpeg` and `ffprobe`.
- No cloud service, account, AI provider setup, marketplace, BYOK, or online registry is required.

这是 Game Sprite Forge 的第一个公开测试版本。

Game Sprite Forge 是一个本地 macOS 工作台，用于把视频、PNG 序列帧、精灵表和 `.gsfpack` 文件夹转换成可用于游戏开发的 2D 精灵动画资产。

## 亮点

- 支持导入视频、PNG 序列帧、精灵表和 `.gsfpack`。
- 精灵表支持固定网格和透明间隔切分。
- 支持背景处理、帧规范化、质量检查和导出就绪状态。
- 可导出 PNG 帧、精灵表、manifest、atlas 元数据、预览素材和 `.gsfpack`。
- 支持本地资源包验证和重新导入。
- 包含 Godot 辅助示例。
